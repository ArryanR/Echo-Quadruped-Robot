# IK Derivations

<code> Distance to the Foot\(r = \sqrt{x^2 + z^2}\) 

\(\cos(\theta_{knee}) = \frac{r^2 - L_1^2 - L_2^2}{2 L_1 L_2}\)\(\theta_{knee} = \arccos\left(\frac{r^2 - L_1^2 - L_2^2}{2 L_1 L_2}\right)\)

Hip AngleThe total hip angle is the sum of the vector angle to the foot and the interior angle of the triangle.Vector angle:
\(\alpha = \arctan2(z, x)\)Interior triangle angle: \(\beta = \arctan2(L_2 \sin(\theta_{knee}), L_1 + L_2 \cos(\theta_{knee}))\)Total Hip Angle: \(\theta_{hip} = \alpha + beta\)
