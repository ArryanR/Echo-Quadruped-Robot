## 🐶 Robot Dog Project

My robot dog was first created in a browser‑based software called **TinkerCAD**.  
It provided basic support, and I started my project there because of its user‑friendly capabilities.

### 🔗 TinkerCAD Model  
https://www.tinkercad.com/things/75EoaUac4LI-robot-dog

### ⚙️ Features
- Servo motors  
- Display screen  
- Ultrasonic distance sensor  
- ESP32‑CAM module  

At the time, coding it was a bit challenging since I wasn’t familiar with Arduino programming.  
However, it became a very important starting point for me.
